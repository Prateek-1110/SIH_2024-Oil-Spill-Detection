# Description

GOD’S EYE – AI-Powered Oil Spill Detection

## Project Overview

This project focuses on building an AI-driven system for detecting oil spills in marine environments by combining vessel tracking data with satellite imagery. It integrates anomaly detection on AIS (Automatic Identification System) data with deep learning–based image segmentation to identify potential spill events accurately and in near real time.

## Key Components

* **AIS-Based Anomaly Detection**
  Machine learning models analyze vessel movement patterns to identify unusual behavior, such as abrupt speed changes, irregular routes, or suspicious loitering, which may indicate illegal discharge activity.

* **Satellite Image Segmentation**
  The system uses DeepLabV3, a state-of-the-art semantic segmentation model, to detect and delineate oil spills from SAR (Synthetic Aperture Radar) imagery. This enables robust detection even under low-visibility conditions like cloud cover or nighttime.

* **Data Fusion Layer**
  Insights from AIS anomaly detection and satellite-based segmentation are combined to improve detection accuracy and reduce false positives.

## Features

* Real-time or near real-time monitoring of marine regions
* Automated detection and localization of oil spills
* Scalable architecture for large-area surveillance
* Modular design for easy integration with maritime monitoring systems
* Supports environmental compliance and regulatory monitoring

## Applications

* Maritime surveillance and safety
* Environmental protection and spill response
* Regulatory enforcement against illegal discharges
* Coastal monitoring agencies and research organizations

## Impact

By automating oil spill detection and integrating multiple data sources, this system reduces response time, minimizes environmental damage, and enhances decision-making for maritime authorities.
